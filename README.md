# Syncleo
Integration of Polaris and Hommyn devices into Home Assistant using UDP protocol

Интеграция для Home Assistant поддерживающая полное управление устройствами экосистем
Polaris IQ Home и Hommyn c использованием протокола UDP

Управление локальное (внутри сети) работает одновременно с облаком и(или) MQTT
